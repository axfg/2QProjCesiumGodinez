Budget Calculator

Purpose and Recommended Use:

This is a tool that aims to subdivide budgets into smaller segments so that people do not overspend on certain categories. It was built with groceries in mind.

Instructions:

To start, specify the amount to be used in the entire budget. Then click the button below to create a budget.
You can create new categories and budgets, as well as adjust them.

To begin, dictate the given budget. This cannot be changed after it is set.
